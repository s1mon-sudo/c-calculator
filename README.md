## Beschreibung

Dieses Projekt ist ein einfaches Schulprojekt, das ich in C++ erstellt habe.  
Es handelt sich um einen Konsolen-Taschenrechner, mit dem grundlegende Rechenoperationen durchgeführt werden können.
